    • Curve Number (CN) Calculator - Area-weighted composite curve number calculations
    • Rational Method C Calculator - Slope-based runoff coefficient calculations  
    • Time of Concentration Calculator - Multi-method implementation (Kirpich, FAA, SCS/NRCS, Kerby)
    • Channel Designer - Interactive trapezoidal channel cross-section design
    
    All tools feature real-time validation, professional output formats, and batch processing capabilities.
